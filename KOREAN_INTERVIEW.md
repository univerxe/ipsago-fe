## 🎯 시스템 개요:

### **사용자 흐름:**
1. 온보딩 완료 (영어로 프로필 입력)

### **AI 면접관이 하는 일:**
- ✅ 한국어로 질문합니다
- ✅ 지원자의 이력서를 검토합니다
- ✅ 채용 공고 요구사항을 기반으로 질문합니다
- ✅ 답변에 따라 후속 질문을 합니다
- ✅ 외국인 지원자를 배려하며 친절하게 대합니다

## 💬 면접 예시:

**AI 면접관:** 안녕하세요! 면접 연습 세션에 오신 것을 환영합니다. 저는 오늘 여러분을 면접할 AI 면접관입니다. 지원자님의 프로필과 채용 공고를 검토했습니다. 먼저 간단히 자기소개를 부탁드립니다.

**지원자:** 안녕하세요. 저는 3년 경력의 프론트엔드 개발자입니다. React와 TypeScript를 주로 사용해왔습니다.

**AI 면접관:** 네, 이력서에서 React 경험을 확인했습니다. React를 사용하여 진행하셨던 프로젝트 중 가장 도전적이었던 프로젝트에 대해 말씀해 주시겠어요?

**지원자:** 실시간 대시보드를 구축했을 때...

**AI 면접관:** 흥미롭습니다! 실시간 데이터를 위한 상태 관리는 어떻게 처리하셨나요? Redux를 사용하셨나요?

...면접 계속

## 🔧 설정 방법:

### 1. OpenAI API 키 받기
https://platform.openai.com/api-keys

### 2. `.env` 파일에 추가
```bash
OPENAI_API_KEY=sk-your-key-here
```

### 3. 서버 재시작
```bash
npm run dev
```

### 4. 테스트
1. `/onboarding` 페이지에서 프로필 작성
2. `/dashboard`에서 채용 공고 확인
3. "Practice" 클릭
4. 한국어로 대화 시작!

## 📋 면접 내용:

### **AI가 질문하는 내용:**

1. **자기소개**
   - 본인 소개
   - 지원 동기

2. **기술 질문**
   - 사용 기술 스택
   - 프로젝트 경험
   - 문제 해결 능력

3. **행동 기반 질문 (STAR 방식)**
   - 어려운 상황 대처
   - 팀워크 경험
   - 성과 사례

4. **마무리**
   - 지원자 질문
   - 다음 단계 안내
